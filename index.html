DIG 345 Site Test

Instructions

Use the content in this repo to create a design, code the page, and publish the site.

  1. Fork this repository to your own Github account.
  2. Create two designs, one desktop and one mobile
    • The design tool you use (e.g. Figma, Photoshop, pencil and paper) should allow you to pre-visualize the layout nearly identical to how it will appear once you code it.
    • Use all the content in this repository, including all the text on this page and files inside images/.
    • You are encouraged to re-organize the text however you like.
    • Feel free to use the Bootstrap Grid Wireframe.
  3. Code the page in HTML and CSS
    • The page should be responsive, using Bootstrap or other responsive framework or method, with breakpoints.
    • Do not use a template for the design. You should be able to code this from scratch.
  4. Add Javascript that
    • Changes content on the page based on some event (a user action, etc.).
    • Logs a message to the console expressing how you feel this test went.
  5. Post two links in the class Slack channel
    • The Figma layout (make sure it is live)
    • The live site, published on Github Pages

Rules

  1. Work on your own.
  2. I will introduce the assignment and then answer questions for a few minutes before we begin.
  3. You can ask me one technical question during the test.
